GTOne ERP 오류 해결 가이드

====================================
1. 로그인/접속 오류
====================================

ERROR_001: "사용자 인증에 실패했습니다"
원인: 잘못된 ID/PW, 계정 잠금, 계정 만료
해결방법:
1. ID/PW 재확인 (대소문자 구분)
2. 5회 이상 실패시 30분 후 재시도
3. 시스템 관리자에게 계정 잠금 해제 요청
4. 비밀번호 재설정: 로그인 화면 > '비밀번호 찾기'

ERROR_002: "세션이 만료되었습니다"
원인: 30분 이상 미사용, 중복 로그인
해결방법:
1. 다시 로그인
2. 다른 PC에서 로그인 확인
3. 브라우저 쿠키 삭제
4. 자동 로그아웃 시간 연장 요청 (보안팀 승인 필요)

ERROR_003: "서버에 연결할 수 없습니다"
원인: 네트워크 오류, 서버 점검, 방화벽
해결방법:
1. 인터넷 연결 상태 확인
2. 회사 VPN 연결 확인
3. 시스템 공지사항 확인 (정기 점검)
4. IT 헬프데스크 문의

====================================
2. 전표/회계 오류
====================================

ERROR_101: "전표번호가 중복되었습니다"
원인: 수동 채번시 중복 입력, 동시 입력
해결방법:
1. 자동 채번 사용 권장
2. 전표번호 조회 후 신규 번호 입력
3. 전표 채번 규칙 확인

ERROR_102: "계정과목이 존재하지 않습니다"
원인: 잘못된 계정코드, 비활성 계정, 삭제된 계정
해결방법:
1. 계정과목 코드 재확인
2. 기준정보 > 계정과목 조회
3. 계정과목 사용여부 'Y' 확인
4. 대체 계정과목 사용

ERROR_103: "마감된 회계기간입니다"
원인: 월/년 마감 완료, 잘못된 전표일자
해결방법:
1. 전표일자 확인 (현재 회계기간)
2. 마감 취소 요청 (승인 필요)
3. 수정 전표로 작성
4. 차기 회계기간에 조정 분개

ERROR_104: "외화 환율이 등록되지 않았습니다"
원인: 해당일자 환율 미등록
해결방법:
1. 기준정보 > 환율관리
2. 해당 일자 환율 등록
3. 한국은행 고시환율 자동 연동 설정
4. 수동 환율 입력

====================================
3. 재고/구매 오류
====================================

ERROR_201: "재고가 부족합니다"
원인: 현재고 부족, 가용재고 부족
해결방법:
1. 재고 현황 조회
2. 미입고 발주 확인
3. 대체 품목 검토
4. 긴급 발주 처리

ERROR_202: "단가 정보가 없습니다"
원인: 단가 미등록, 유효기간 만료
해결방법:
1. 기준정보 > 단가관리
2. 품목별 매입단가 등록
3. 단가 유효기간 확인
4. 거래처별 단가 협의

====================================
4. 결재/워크플로우 오류
====================================

ERROR_301: "결재선이 지정되지 않았습니다"
원인: 결재선 미설정, 조직 변경
해결방법:
1. 결재선 관리에서 설정
2. 부서별 기본 결재선 확인
3. 임시 결재선 지정
4. 대결자 지정

ERROR_302: "결재 권한이 없습니다"
원인: 권한 부족, 대결 기간 만료
해결방법:
1. 결재 권한 확인
2. 대결자 설정 확인
3. 원결재자에게 처리 요청
4. 결재 권한 재설정 요청

====================================
5. 보고서/출력 오류
====================================

ERROR_401: "보고서 생성에 실패했습니다"
원인: 대용량 데이터, 메모리 부족
해결방법:
1. 조회 기간 축소
2. 조회 조건 세분화
3. 엑셀 다운로드 대신 CSV 사용
4. 야간 배치로 생성 요청

ERROR_402: "프린터 연결 오류"
원인: 프린터 드라이버, 네트워크
해결방법:
1. 프린터 상태 확인
2. PDF로 저장 후 출력
3. 다른 프린터 선택
4. 프린터 드라이버 재설치

====================================
6. 성능/속도 문제
====================================

SLOW_001: "조회 속도가 느립니다"
원인: 대용량 데이터, 복잡한 조건
해결방법:
1. 조회 조건 단순화
2. 인덱스 생성 요청
3. 필수 항목만 조회
4. 배치 조회 활용

SLOW_002: "화면 전환이 느립니다"
원인: 브라우저 캐시, 네트워크
해결방법:
1. 브라우저 캐시 삭제
2. 쿠키 삭제
3. 다른 브라우저 사용
4. 유선 네트워크 사용
