GTOne ERP 자주 묻는 질문 (FAQ)

Q1. 로그인이 되지 않습니다.
A: 다음 사항을 확인해 주세요:
   1) 아이디와 비밀번호 대소문자 확인
   2) Caps Lock 키 확인
   3) 계정 잠금 여부 확인 (5회 실패시 자동 잠금)
   4) 비밀번호 유효기간 확인 (90일)
   해결이 안 될 경우 시스템 관리자에게 문의하세요.

Q2. 전표 입력 시 "권한이 없습니다" 메시지가 나타납니다.
A: 해당 계정과목이나 부서에 대한 입력 권한이 없는 경우입니다.
   - 시스템 > 내 권한 조회에서 현재 권한 확인
   - 필요시 상급자에게 권한 요청
   - 임시로 다른 사용자에게 입력 요청 가능

Q3. 재무제표 수치가 맞지 않습니다.
A: 다음 순서로 확인하세요:
   1) 조회 기간 설정 확인
   2) 미승인 전표 존재 여부 확인
   3) 결산 작업 완료 여부 확인
   4) 계정 재분류 내역 확인
   5) 시산표와 대조 확인

Q4. 엑셀 다운로드가 안 됩니다.
A: 브라우저 팝업 차단 설정을 확인하세요.
   - Chrome: 주소창 오른쪽 팝업 차단 아이콘 클릭
   - Edge: 설정 > 쿠키 및 사이트 권한 > 팝업
   - 회사 보안 프로그램 설정 확인

Q5. 거래처 등록은 어떻게 하나요?
A: 기준정보 > 거래처관리 > 거래처등록
   필수 입력 정보:
   - 사업자번호 (자동 중복 체크)
   - 거래처명
   - 대표자명
   - 업태/종목
   - 주소
   - 담당자 연락처

Q6. 전표를 잘못 입력했습니다. 수정이 가능한가요?
A: 전표 상태에 따라 다릅니다:
   - 임시저장 상태: 자유롭게 수정 가능
   - 승인 전: 수정 후 재승인 요청
   - 승인 완료: 수정 전표 또는 역분개 전표 생성
   - 마감 후: 시스템 관리자에게 마감 취소 요청

Q7. 부가세 신고 자료는 어디서 확인하나요?
A: 재무회계 > 부가세 > 부가세신고서
   - 예정/확정 신고 구분 선택
   - 신고 기간 확인
   - 매입/매출 내역 검증
   - 전자신고 파일 생성

Q8. 급여 정보가 회계에 반영되지 않습니다.
A: HR 모듈과 회계 모듈 간 인터페이스 확인:
   1) HR 모듈에서 급여 확정 처리 확인
   2) 인터페이스 > 급여전표생성 실행
   3) 생성된 전표 확인 및 승인
   4) 계정 매핑 정보 확인

Q9. 예산 대비 실적을 어떻게 확인하나요?
A: 재무회계 > 예산관리 > 예산실적분석
   - 부서별/프로젝트별 선택
   - 월별/누계 조회 옵션
   - 초과 항목 자동 표시
   - 예산 소진율 그래프 제공

Q10. 시스템이 느립니다.
A: 다음을 확인하세요:
   1) 인터넷 연결 상태
   2) 동시 접속자 수 (오전 9-10시 집중)
   3) 대용량 데이터 조회 여부
   4) 브라우저 캐시 삭제
   5) 권장 브라우저 사용 (Chrome, Edge 최신 버전)
