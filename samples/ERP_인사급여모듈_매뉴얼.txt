GTOne ERP 인사급여 모듈 사용자 매뉴얼 v2.0

==================================================
1. 인사관리
==================================================

1.1 직원 정보 관리
메뉴 경로: 인사급여 > 인사관리 > 직원정보

직원 등록 절차:
1) [신규] 버튼 클릭
2) 기본 정보 입력
   - 사번: 자동 채번 (YYMMDDxxx 형식)
   - 성명 (한글/영문)
   - 주민등록번호 (암호화 저장)
   - 입사일자
   - 부서/직급/직책
3) 상세 정보 입력
   - 연락처 (휴대폰/이메일)
   - 주소 (자택/현거주지)
   - 학력사항
   - 경력사항
   - 자격증
4) 급여 정보 설정
   - 급여형태 (월급/연봉/시급)
   - 기본급
   - 수당 항목
   - 공제 항목
5) 4대보험 설정
   - 국민연금 등급
   - 건강보험 등급
   - 고용보험 가입여부
   - 산재보험 적용

인사 발령 처리:
1) 인사관리 > 인사발령
2) 발령 구분 선택
   - 부서이동
   - 승진/승급
   - 직책변경
   - 휴직/복직
   - 퇴직
3) 발령일자 및 사유 입력
4) 결재선 지정
5) 발령 후 처리사항
   - 조직도 자동 반영
   - 권한 재설정
   - 급여 정보 갱신

1.2 근태 관리
메뉴 경로: 인사급여 > 근태관리 > 근태현황

일일 근태 등록:
- 출근: 카드 태그 또는 모바일 앱
- 퇴근: 카드 태그 또는 모바일 앱
- 외출/복귀: 사유 입력 필수
- 출장: 사전 신청 필요

근태 이상자 처리:
- 지각: 월 3회 초과시 경고
- 조퇴: 사유서 제출
- 결근: 무단결근시 징계 대상
- 연차: 사전 신청 원칙

초과근무 관리:
1) 사전 신청
   - 초과근무 예정시간
   - 업무 내용
   - 상급자 승인
2) 실적 등록
   - 실제 근무시간
   - 수행 업무
3) 수당 계산
   - 평일: 시급 × 1.5
   - 휴일: 시급 × 2.0
   - 야간: 시급 × 0.5 추가

==================================================
2. 급여관리
==================================================

2.1 급여 계산
메뉴 경로: 인사급여 > 급여관리 > 급여계산

월 급여 계산 프로세스:
1) 근태 마감 확인
   - 근태 데이터 검증
   - 이상 근태 확인
   - 휴가 사용 반영
2) 급여 기초자료 생성
   - 기본급
   - 초과수당
   - 각종 수당
3) 공제 항목 계산
   - 소득세
   - 주민세
   - 4대보험료
   - 기타 공제
4) 급여 시뮬레이션
   - 계산 결과 검토
   - 오류 사항 수정
5) 급여 확정
   - 최종 승인
   - 급여명세서 생성

특수 급여 처리:
- 상여금: 분기/반기/연간
- 성과급: 평가 등급 연동
- 퇴직금: 평균임금 × 근속연수
- 연차수당: 미사용 연차 보상

2.2 급여 지급
메뉴 경로: 인사급여 > 급여관리 > 급여지급

급여 이체 처리:
1) 은행 이체 파일 생성
   - 급여 이체 명세
   - 은행별 포맷 지원
   - 암호화 전송
2) 이체 결과 확인
   - 성공/실패 건수
   - 실패 사유 확인
   - 재처리

급여명세서 발송:
- 이메일 자동 발송
- 모바일 앱 조회
- 급여명세서 출력 (요청시)

==================================================
3. 연말정산
==================================================

3.1 연말정산 준비
메뉴 경로: 인사급여 > 연말정산 > 기초자료수집

자료 수집 항목:
- 부양가족 정보
- 보험료 납입증명서
- 의료비 납입내역
- 교육비 납입증명서
- 기부금 영수증
- 신용카드 사용내역
- 주택자금 공제

근로자 소득공제 신고서:
1) 온라인 입력
   - 공제 항목별 입력
   - 증빙 서류 업로드
   - 임시 저장 기능
2) 서류 제출
   - 원본 제출 항목
   - 제출 기한 안내

3.2 연말정산 처리
메뉴 경로: 인사급여 > 연말정산 > 정산처리

정산 프로세스:
1) 자료 검증
   - 공제 한도 확인
   - 중복 공제 체크
   - 증빙 서류 확인
2) 세액 계산
   - 과세표준 산출
   - 산출세액 계산
   - 세액공제 적용
3) 정산 결과
   - 추가 납부/환급
   - 정산 내역서 생성
4) 신고/납부
   - 원천세 신고
   - 지급명세서 제출

4. 복리후생
==================================================

4.1 휴가 관리
메뉴 경로: 인사급여 > 복리후생 > 휴가관리

연차 휴가:
- 발생: 1년 15일 (근속년수 추가)
- 사용: 사전 신청 원칙
- 촉진: 미사용시 사용 촉구
- 보상: 미사용 연차 수당 지급

특별 휴가:
- 경조사 휴가
  * 본인 결혼: 5일
  * 자녀 결혼: 1일
  * 배우자 출산: 10일
  * 부모 사망: 5일
- 포상 휴가
- 하계 휴가

4.2 복리후생 프로그램
메뉴 경로: 인사급여 > 복리후생 > 복지포인트

복지카드 관리:
- 포인트 지급 (분기별)
- 사용처 제한
- 잔액 조회
- 이월 정책

사내 복지:
- 건강검진 지원
- 자녀 학자금
- 경조금 지급
- 동호회 지원

==================================================
5. 조직 관리
==================================================

5.1 조직도 관리
메뉴 경로: 인사급여 > 조직관리 > 조직도

조직 구조:
- 본부 > 부서 > 팀 > 파트
- 직급 체계: 사원-대리-과장-차장-부장
- 직책 체계: 팀원-파트장-팀장-부서장

조직 변경:
1) 조직 개편 시뮬레이션
2) 변경 영향도 분석
3) 인사 발령 연동
4) 권한 재설정

5.2 직무 관리
메뉴 경로: 인사급여 > 조직관리 > 직무관리

직무 분류:
- 직군: 관리직/영업직/기술직
- 직무: 세부 업무 구분
- 직무 기술서 관리
- 필요 역량 정의

인력 현황:
- 부서별 인원 현황
- 직급별 분포
- 연령대별 분석
- 근속년수 분석

6. 자주 발생하는 문제 해결
==================================================

Q: 급여명세서가 이메일로 오지 않습니다.
A: 인사정보에서 이메일 주소 확인
   스팸메일함 확인
   IT부서에 메일 서버 확인 요청

Q: 연차가 잘못 계산되었습니다.
A: 입사일자 확인
   중간 입사자는 월할 계산
   전년도 사용 내역 확인
   인사팀 문의

Q: 4대보험료가 변경되었습니다.
A: 매년 보험료율 변경
   소득 구간 변경
   부양가족 변동
   고지서 확인
